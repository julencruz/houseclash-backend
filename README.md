# HouseClash Backend
Backend REST API forHouseClash: A gamified household chore manager built with Kotlin, Spring Boot, and Hexagonal Architecture.
Central orchestrator for collaborative household chore management. Developed in Kotlin and Spring Boot applying Hexagonal Architecture to isolate the domain logic. It exposes a REST API that manages the task lifecycle, the dynamic market (Kudos), and the gamification engine (gacha system)
